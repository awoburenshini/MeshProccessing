=============================================================================
== libTinyTIFF 
==    is a library for generating Matlab 5.0 MAT-files                     ==
=============================================================================

This is a lightweight C/C++ library, which is able to read and write basic TIFF
files. It is significantly faster than libTIFF, especially in writing large
multi-frame TIFFs.
